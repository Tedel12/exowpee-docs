# Common errors

## Error response format
```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid Email",
    "details": {
      "email": ["Invalid format"]
    }
  }
}
```

## Main codes

| Code | Type | Cause | Solution |
|------|------|-------|----------|
| `400` | Validation | Missing fields | Check the spec |
| `401` | Unauthorized | Invalid token | Redo login |
| `403` | Forbidden | No permission | Check your roles |
| `404` | Not Found | ID does not exist | Check ID |
| `429` | Rate Limited | Too many requests | Wait for `Retry-After` |
| `500` | ServerError | Server bug | Support contact |

## Useful headers
```
X-Request-ID: req_abc123 ← For support
Retry-After: 60 ← Second before retry
