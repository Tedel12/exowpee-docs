The Exowpee API uses **Bearer Token (JWT)** authentication to secure all endpoints.

## 1. Obtain a token

**Endpoint**: `POST /v1/auth/login`

### Query
```bash
curl -X POST https://api.exowpee.com/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@exowpee.com",
    "password": "your-password"
  }'
```
### Success response (200)
```json
{
  "authToken": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
  "user": {
    "id": 1,
    "name": "John Doe",
    "email": "john@exowpee.com",
    "companyId": 1
  }
}
```

### Common errors
| Code | Message | Solution |
|------|---------|----------|
| 400 | Invalid credentials | Check email/password |
| 429 | Too many requests | Wait before trying again |

## 2. Use the token

Add the `Authorization` header to **each request**:

```
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...
```

### Exemples

**cURL**
```bash
curl -X GET https://api.exowpee.com/v1/catalog/categories \
  -H "Authorization: Bearer VOTRE_TOKEN"
```

**JavaScript**
```javascript
const token = "VOTRE_TOKEN";

fetch("https://api.exowpee.com/v1/catalog/categories", {
  method: "GET",
  headers: {
    "Authorization": `Bearer ${token}`,
    "Content-Type": "application/json"
  }
})
.then(res => res.json())
.then(data => console.log(data));
```
**Python**
```python
import requests

headers = {
    "Authorization": "Bearer YOUR_TOKEN"
}

response = requests.get(
    "https://api.exowpee.com/v1/catalog/categories",
    headers=headers
)
print(response.json())
```

## 3. User Info

Retrieve your information: `GET /v1/auth/me`

```bash
curl -X GET https://api.exowpee.com/v1/auth/me \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Security

- **Never share** your token
- **Token expires** → re-login
- **Use HTTPS** in production
