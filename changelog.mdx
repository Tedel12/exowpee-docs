# Changelog

## [1.1.0] - February 12, 2026

### Added
- `POST /v1/order/{id}/confirm`
- `POST /v1/order/{id}/reject` 
- `GET /v1/reports/sales-per-product`
- Webhook support (`order.created`, `stock.low`)

### Changed
- `stock.regularisation` timeout increased to 30s

## [1.0.0] - January 1, 2026

### Added
- Initial API endpoints
- Bearer JWT authentication
- Rate limiting (100 req/min)
